FORMULÁRIO DE ENVIO DE EVIDÊNCIAS

INSTITUTO BR ARTE

Este formulário deve ser preenchido uma vez para cada atividade executada. Não coloque várias oficinas, reuniões, aulas ou ações no mesmo envio. Cada atividade precisa ter data, local, descrição e evidências próprias. Se uma evidência não existir, informe "Pendente". Se não se aplicar à atividade, informe "Não se aplica".

Identificação do Responsável

Dados da Atividade Executada

Informe o dia em que a atividade aconteceu.

Informe a hora em que começou, se souber.

Informe a hora em que terminou, se souber.

Escreva o local onde a atividade aconteceu. Exemplo: Galpão do Muro, Maré.

Escolha o tipo que mais combina com a atividade ou evidência registrada.

Se souber a rubrica, escreva aqui. Se não souber, deixe em branco. A equipe técnica classificará depois.

Descreva o perfil das pessoas que participaram — não o número. Exemplos: mulheres empreendedoras, jovens do território, moradores da Maré, participantes inscritos na oficina, equipe técnica do projeto. A quantidade de pessoas fica no campo "Quantidade de participantes".

Explique em poucas linhas o que aconteceu, quem participou e qual foi a entrega realizada.

3

Evidências da Execução

Os campos de evidência não devem ficar em branco. Cole o link quando existir. Escreva "Pendente" quando o documento ainda não estiver pronto. Escreva "Não se aplica" quando aquele documento não fizer sentido para esta atividade.

Cole o link da pasta principal onde estão todos os arquivos desta atividade. Se não tiver ainda, escreva "Pendente".

Cole o link da pasta ou arquivo com as fotos. Se ainda não tiver, escreva "Pendente".

Cole o link dos vídeos. Se não tiver vídeo, escreva "Não se aplica". Se ainda vai enviar, escreva "Pendente".

Cole o link da lista de presença. Se a atividade não tiver lista, escreva "Não se aplica". Se ainda vai enviar, escreva "Pendente".

Cole o link dos certificados. Se a atividade não gera certificado, escreva "Não se aplica". Se ainda vai enviar, escreva "Pendente".

Cole o link do material usado na atividade. Se não houver material, escreva "Não se aplica".

Cole o link do card de divulgação. Se não teve card, escreva "Não se aplica".

Se já existir um relatório da atividade, cole o link. Se não tiver, pode deixar em branco.

Cole links de prints, mensagens, divulgações ou registros de comunicação. Se não tiver, escreva "Não se aplica".

Cole outros links importantes. Se não houver outros documentos, escreva "Não se aplica".
